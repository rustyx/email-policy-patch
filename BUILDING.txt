
Requires: JDK 6+, Maven 3+

Prerequisites: Smali 1.4+.

 * Build smali for Java 1.6
   cd ..; git clone https://code.google.com/p/smali
   Edit smali/build.gradle and add this to section subprojects { }:
    sourceCompatibility='1.6'
    targetCompatibility='1.6'


 * Install smali.jar and baksmali.jar into the local Maven repo:
   mvn install:install-file -Dpackaging=jar -DgroupId=org.jf -DartifactId=smali -Dversion=dev -Dfile=smali.jar
   mvn install:install-file -Dpackaging=jar -DgroupId=org.jf -DartifactId=baksmali -Dversion=dev -Dfile=baksmali.jar

 * Run build.cmd
   (will create /target/email-policy-patch-x.x.zip)
